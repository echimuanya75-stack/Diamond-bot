# Diamond WhatsApp Bot

Diamond Bot is a simple WhatsApp bot built with Node.js and Baileys. It responds to commands and promotes the official WhatsApp channel.

## Commands

- `menu` - Show the bot menu  
- `channel` - Join the official WhatsApp channel  
- `help` - Show help message  

## Deployment

This bot can be deployed on **Render**, **Railway**, or any Node.js host.

### Render Setup

1. Fork this repository to your GitHub account
2. Go to [Render](https://render.com) → New Web Service → Connect GitHub Repo
3. Build Command: `npm install`
4. Start Command: `npm start`
5. Deploy the project
6. Pair your WhatsApp when the bot generates a QR code
